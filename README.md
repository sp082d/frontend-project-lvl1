## Как запустить?

### Установка
```
$ npm install
$ npm link
```

`npm link` нужно запускать в корневой директории проекта

### Запуск игор
Шаблон для запуска: `brain-`games/even/prime/gcd/calc/progression 
```
$ brain-games
```
